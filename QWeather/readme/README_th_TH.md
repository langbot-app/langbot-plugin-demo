# QWeather

ปลั๊กอิน [LangBot](https://github.com/langbot-app/langbot) สำหรับแสดงข้อมูลสภาพอากาศโดยใช้ QWeather API

โค้ดดัดแปลงมาจาก [nonebot-plugin-heweather](https://github.com/kexue-z/nonebot-plugin-heweather)

## คุณสมบัติ

- แสดงข้อมูลสภาพอากาศในรูปแบบข้อความ
- รองรับ QWeather API (การสมัครสมาชิกแบบฟรี, มาตรฐาน และเชิงพาณิชย์)
- แสดงสภาพอากาศปัจจุบัน, คุณภาพอากาศ, พยากรณ์อากาศ, การแจ้งเตือน และเวลาพระอาทิตย์ขึ้น/ตก
- รองรับการพยากรณ์อากาศล่วงหน้าหลายวัน

## การกำหนดค่า

ก่อนใช้ปลั๊กอินนี้ คุณต้อง:

1. ลงทะเบียนและรับ API key จาก [QWeather](https://dev.qweather.com/)
2. กำหนดค่าปลั๊กอินใน LangBot WebUI:
   - **QWeather API Key**: API key ของคุณจาก QWeather
   - **API Type**: เลือกประเภทการสมัครสมาชิกของคุณ (Free/Standard/Commercial)

## การใช้งาน

ส่งคำสั่งต่อไปนี้เพื่อรับข้อมูลสภาพอากาศ:

```
!weather <ชื่อเมือง>
```
