#แจ้งกำหนดการ

กำหนดเวลาการแจ้งเตือนด้วยภาษาที่เป็นธรรมชาติ

## คุณสมบัติ

ScheNotify เป็นปลั๊กอิน LangBot ที่ช่วยให้ผู้ใช้สามารถตั้งค่าการแจ้งเตือนแบบกำหนดเวลาผ่านการโต้ตอบด้วยภาษาธรรมชาติกับ LLM

### คุณสมบัติหลัก

-**การโต้ตอบด้วยภาษาธรรมชาติ**: ทำความเข้าใจความตั้งใจในการจัดกำหนดการของผู้ใช้ผ่าน LLM
-**การแยกวิเคราะห์เวลาอัจฉริยะ**: รับเวลาปัจจุบันโดยอัตโนมัติและคำนวณเวลาเตือนความจำ
-**รองรับหลายภาษา**: รองรับข้อความเตือนความจำภาษาจีนและอังกฤษ
-**คำสั่งการจัดการกำหนดการ**: ดูและลบการแจ้งเตือนตามกำหนดเวลา
-**การแจ้งเตือนอัตโนมัติ**: ส่งข้อความเตือนความจำอัตโนมัติตามเวลาที่กำหนด

## การกำหนดค่า

### การตั้งค่าภาษา

คุณสามารถเลือกภาษาสำหรับข้อความเตือนได้ในการกำหนดค่าปลั๊กอิน:

-`zh_Hans`(จีนตัวย่อ) - ค่าเริ่มต้น
-`en_US`(อังกฤษ)

## การใช้งาน

### 1. กำหนดเวลาผ่าน LLM

เพียงบอก LLM กำหนดการของคุณเป็นภาษาธรรมชาติ:

**ตัวอย่าง:**
```
Remind me to have a meeting at 3 PM tomorrow
Remind me to submit the report at 9 AM the day after tomorrow
Remind me to have lunch at 12 PM next Monday
Remind me about Christmas dinner at 2024-12-25 18:00
```

LLM จะ: โดยอัตโนมัติ
1. โทร`get_current_time_str`เพื่อดูเวลาปัจจุบัน
2. แยกนิพจน์เวลาของคุณและแปลงเป็นรูปแบบมาตรฐาน
3. โทร`schedule_notify`เพื่อสร้างการแจ้งเตือน

### 2. ดูการแจ้งเตือนตามกำหนดเวลา

ใช้คำสั่งเพื่อดูการแจ้งเตือนตามกำหนดเวลาทั้งหมด:

```
!sche
```

ตัวอย่างผลลัพธ์:
```
[Notify] Scheduled reminders:
#1 2024-12-25 18:00:00: Christmas dinner
#2 2024-12-26 09:00:00: Submit report
```

### 3. ลบการแจ้งเตือน

ใช้คำสั่งเพื่อลบการเตือนความจำเฉพาะ (โดยใช้ตัวเลขจาก`!sche`):

```
!dsche i <number>
```

ตัวอย่าง:
```
!dsche i 1   # Delete the 1st reminder
```

## ส่วนประกอบ

### เครื่องมือ

1.**get_current_time_str**- รับเวลาปัจจุบัน
   - รูปแบบการส่งคืน:`YYYY-MM-DD HH:MM:SS`
   - LLM ต้องเรียกใช้เครื่องมือนี้ก่อนที่จะตั้งค่าการแจ้งเตือน

2.**schedule_notify**- การแจ้งเตือนกำหนดการ
   - พารามิเตอร์: สตริงเวลา ข้อความเตือนความจำ
   - รับข้อมูลเซสชันโดยอัตโนมัติจากพารามิเตอร์เซสชันของเครื่องมือ

### คำสั่ง

1.**sche**(นามแฝง: s) - แสดงรายการการแจ้งเตือนที่กำหนดเวลาไว้ทั้งหมด
2.**dsche**(นามแฝง: d) - ลบการแจ้งเตือนที่ระบุ

## รายละเอียดทางเทคนิค

- ตรวจสอบช่วงเวลา: ทุกๆ 60 วินาที
- ความแม่นยำของเวลา: ระดับนาที (ตรวจสอบทุกนาที)
- ข้อมูลเซสชัน: รับโดยอัตโนมัติผ่านพารามิเตอร์เซสชันของเครื่องมือ
- ความคงอยู่: ปัจจุบันใช้ที่เก็บข้อมูลในหน่วยความจำ (หายไปเมื่อรีสตาร์ท)

## ตัวอย่างการสนทนา

**ผู้ใช้:**เตือนให้ฉันเข้าร่วมการประชุมเวลา 14.00 น. ในวันพรุ่งนี้

**LLM:**แน่นอน ฉันจะตั้งการช่วยเตือนให้คุณ

*[LLM โทร get_current_time_str]*
*[LLM โทร schedule_notify(time_str="2024-12-26 14:00:00", message="Attend conference")]*

**LLM:**เสร็จแล้ว! ฉันจะเตือนคุณที่ 2024-12-26 14:00:00: เข้าร่วมการประชุม

*[วันถัดไป เวลา 14.00 น.]*

**Bot:**[แจ้งเตือน] เข้าร่วมการประชุม

## หมายเหตุ

- เวลาเตือนจะต้องเป็นเวลาในอนาคต เวลาที่ผ่านมาจะถูกปฏิเสธ
- ข้อความเตือนจะถูกส่งไปยังเซสชันเดียวกับที่ตั้งการแจ้งเตือนไว้
- การแจ้งเตือนที่ยังไม่ได้ส่งจะหายไปหลังจากรีสตาร์ทปลั๊กอิน (ความคงอยู่จะได้รับการสนับสนุนในเวอร์ชันต่อ ๆ ไป)
