# EssentialCommands

以前 LangBot に含まれていた必須コマンドは、以下のコマンドを提供します：

- `!cmd` - 登録されているすべてのコマンドを表示
- `!help` - コマンドのヘルプを表示
- `!reset` - 現在のセッションをリセット
- `!version` - LangBot のバージョンを表示
- `!func` - 利用可能なすべての LLM ツールを表示
- `!plugin` - 読み込まれているすべてのプラグインを表示

このプラグインでは、以前 LangBot で利用可能だった一部のコマンドが削除されています。これらのコマンドが必要な場合は、issue を作成してください。
