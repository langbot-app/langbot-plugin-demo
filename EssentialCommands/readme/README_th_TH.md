#คำสั่งสำคัญ

คำสั่งสำคัญซึ่งก่อนหน้านี้รวมอยู่ใน LangBot มีคำสั่งด้านล่าง:

-`!cmd`- แสดงคำสั่งที่ลงทะเบียนไว้ทั้งหมด
-`!help`- แสดงความช่วยเหลือของคำสั่ง
-`!reset`- รีเซ็ตเซสชันปัจจุบัน
-`!version`- แสดงเวอร์ชันของ LangBot
-`!func`- แสดงเครื่องมือ LLM ที่มีอยู่ทั้งหมด
-`!plugin`- แสดงปลั๊กอินที่โหลดทั้งหมด

ปลั๊กอินนี้จะลบคำสั่งบางคำสั่งที่เคยมีใน LangBot หากคุณต้องการคำสั่งเหล่านี้ โปรดแจ้งปัญหา
