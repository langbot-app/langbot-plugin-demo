#Các lệnh cần thiết

Các lệnh thiết yếu, trước đây có trong LangBot, cung cấp các lệnh bên dưới:

-`!cmd`- Hiển thị tất cả lệnh đã đăng ký
-`!help`- Hiển thị sự trợ giúp của lệnh
-`!reset`- Đặt lại phiên hiện tại
-`!version`- Hiển thị phiên bản LangBot
-`!func`- Hiển thị tất cả các công cụ LLM có sẵn
-`!plugin`- Hiển thị tất cả các plugin đã tải

Plugin này loại bỏ một số lệnh trước đây có sẵn trong LangBot, nếu bạn cần những lệnh này, vui lòng gửi vấn đề.
