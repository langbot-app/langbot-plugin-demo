#Comandosesenciales

Los comandos esenciales, incluidos anteriormente en LangBot, proporcionan los siguientes comandos:

-`!cmd`- Mostrar todos los comandos registrados
-`!help`- Muestra la ayuda de un comando
-`!reset`- Restablecer la sesión actual
-`!version`- Mostrar versión de LangBot
-`!func`- Mostrar todas las herramientas LLM disponibles
-`!plugin`- Mostrar todos los complementos cargados

Este complemento elimina algunos comandos que estaban disponibles anteriormente en LangBot; si necesita estos comandos, presente un problema.
