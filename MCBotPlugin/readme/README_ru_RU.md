# MCBotPlugin

Плагин LangBot для чат-групп серверов Minecraft: привязка сервера к группе, запрос состояния сервера и онлайн-игроков в реальном времени, а также учёт игрового времени каждого игрока.

> Это перенос старого [MCBotPlugin](https://github.com/langbot-app/MCBotPlugin) (созданного для QChatGPT) на новый SDK плагинов LangBot. Хранилище MongoDB заменено встроенным KV-хранилищем плагина (внешняя база данных не требуется), синхронный запрос состояния Minecraft через `mctools` заменён асинхронным `mcstatus`, а фоновая выборка игрового времени переведена с потоков на задачу asyncio.

## Возможности

- **Привязка сервера**: каждая группа может привязать один сервер Minecraft (Java Edition)
- **Запрос состояния**: MOTD, версия, число игроков онлайн и список игроков в реальном времени
- **Статистика игрового времени**: фоновая задача периодически опрашивает онлайн-игроков и суммирует онлайн-время каждого игрока за любой период

## Команды

| Команда | Описание | Права |
| --- | --- | --- |
| `!mcbot` | Показать справку | Все |
| `!mcbot bind <адрес[:порт]>` | Привязать сервер к этой группе | Администратор |
| `!mcbot unbind` | Отвязать сервер | Администратор |
| `!mcbot status` | Показать состояние сервера и онлайн-игроков | Все |
| `!mcbot time [минуты]` | Показать статистику игрового времени (по умолчанию 1440 мин = 24 ч) | Все |

> Администраторы определяются настройкой `admins` в LangBot (`{launcher_type}_{launcher_id}`).

## Настройки

| Ключ | Описание | По умолчанию |
| --- | --- | --- |
| `track_interval` | Интервал выборки онлайн-игроков в секундах (мин. 15) | 60 |
| `ping_timeout` | Таймаут пинга сервера в секундах | 10 |

## Зависимости

- [`mcstatus`](https://github.com/py-mine/mcstatus) — Запрос состояния серверов Minecraft

## Хранение данных

Привязки и записи об онлайне хранятся во встроенном KV-хранилище плагина LangBot; MongoDB не требуется. Записи об онлайне по умолчанию хранятся 14 дней.
