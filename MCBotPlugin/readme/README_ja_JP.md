# MCBotPlugin

Minecraft サーバーのチャットグループ向け LangBot プラグイン：サーバーをグループにバインドし、サーバーの稼働状況やオンラインプレイヤーを照会し、プレイヤーごとのプレイ時間を集計します。

> これは旧版 [MCBotPlugin](https://github.com/langbot-app/MCBotPlugin)（QChatGPT ベース）を新しい LangBot プラグイン SDK へ移行したバージョンです。ストレージは MongoDB からプラグイン内蔵の KV ストレージ（外部データベース不要）に変更され、Minecraft の状態照会は同期 `mctools` から非同期 `mcstatus` に、バックグラウンドのプレイ時間サンプリングはスレッドから asyncio タスクに変更されています。

## 機能

- **サーバーのバインド**：グループごとに 1 つの Minecraft（Java 版）サーバーをバインドできます
- **状態照会**：MOTD、バージョン、オンライン人数、プレイヤー一覧をリアルタイムで表示
- **プレイ時間集計**：バックグラウンドタスクがオンラインプレイヤーを定期的にサンプリングし、任意の期間のプレイヤーごとのオンライン時間を集計します

## コマンド

| コマンド | 説明 | 権限 |
| --- | --- | --- |
| `!mcbot` | ヘルプを表示 | 全員 |
| `!mcbot bind <アドレス[:ポート]>` | サーバーをこのグループにバインド | 管理者 |
| `!mcbot unbind` | サーバーのバインドを解除 | 管理者 |
| `!mcbot status` | サーバーの状態とオンラインプレイヤーを表示 | 全員 |
| `!mcbot time [分]` | プレイ時間統計を表示（デフォルト 1440 分 = 24 時間） | 全員 |

> 管理者は LangBot の `admins` 設定（`{launcher_type}_{launcher_id}`）で決まります。

## 設定項目

| 設定 | 説明 | デフォルト |
| --- | --- | --- |
| `track_interval` | オンラインプレイヤーをサンプリングする間隔（秒）、最小 15 | 60 |
| `ping_timeout` | サーバー Ping のタイムアウト（秒） | 10 |

## 依存関係

- [`mcstatus`](https://github.com/py-mine/mcstatus) —— Minecraft サーバーの状態照会

## データストレージ

バインド情報とオンライン記録は LangBot のプラグイン内蔵 KV ストレージに保存され、MongoDB は不要です。オンライン記録はデフォルトで 14 日間保持されます。
