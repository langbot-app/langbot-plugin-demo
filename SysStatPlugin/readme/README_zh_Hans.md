# SysStatPlugin

[LangBot](https://github.com/langbot-app/langbot) 的系统状态查看插件，可查看 CPU、内存和磁盘使用情况。

参考 [sysstatqcbot](https://github.com/Soulter/sysstatqcbot) 制作

## 功能特性

- 查看当前进程内存占用
- 查看系统内存信息（总量、已用、空闲、使用率）
- 查看 CPU 信息（用户态时间、系统态时间、空闲时间、使用率）
- 查看 CPU 核心数和频率
- 查看磁盘使用信息

## 使用方法

向机器人发送以下命令之一：

```
!sysstat
```

机器人将回复当前系统状态信息。
