# SysStatPlugin

CPU、メモリ、ディスク使用量などのシステムステータスを表示するための [LangBot](https://github.com/langbot-app/langbot) プラグイン。

[sysstatqcbot](https://github.com/Soulter/sysstatqcbot) からインスピレーションを受けました

＃＃ 特徴

- 現在のプロセスメモリ使用量を表示
- システム メモリ情報の表示 (合計、使用済み、空き、使用率)
- CPU 情報の表示 (ユーザー時間、システム時間、アイドル時間、使用率)
- CPU コア数と周波数を表示
- ディスク使用量情報の表示

＃＃ 使用法

次のコマンドのいずれかをボットに送信します。

```
!sysstat
```

ボットは現在のシステム ステータス情報を返信します。
