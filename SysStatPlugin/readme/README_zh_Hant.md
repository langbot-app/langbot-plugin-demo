# SysStatPlugin

[LangBot](https://github.com/langbot-app/langbot) 插件，用於查看包括 CPU、記憶體和磁碟使用情況在內的系統狀態。

靈感來自 [sysstatqcbot](https://github.com/Soulter/sysstatqcbot)

## 功能

- 查看當前進程記憶體佔用
- 查看系統記憶體資訊（總量、已用、空閒、使用率）
- 查看 CPU 資訊（用戶時間、系統時間、空閒時間、使用率）
- 查看 CPU 核心數和頻率
- 查看磁碟使用情況資訊

## 使用方法

向機器人發送以下命令之一：

```
!sysstat
```

機器人將回覆當前的系統狀態資訊。
