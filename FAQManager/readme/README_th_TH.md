# ตัวจัดการ FAQ

จัดการรายการคำถามที่พบบ่อยผ่านหน้าเว็บใน WebUI ของ LangBot และให้ LLM ค้นหาได้ระหว่างการสนทนา

## คุณสมบัติ

- **คอมโพเนนต์ Page**: อินเทอร์เฟซ CRUD สำหรับจัดการคู่คำถาม-คำตอบ เข้าถึงได้จากส่วน "หน้าปลั๊กอิน" ในแถบด้านข้าง
- **คอมโพเนนต์ Tool**: `search_faq` — ให้ LLM ค้นหาฐานข้อมูล FAQ ด้วยคำสำคัญและส่งคืนรายการที่ตรงกันให้ผู้ใช้
- **จัดเก็บถาวร**: รายการ FAQ จะถูกบันทึกผ่านที่เก็บข้อมูลปลั๊กอินและไม่สูญหายเมื่อรีสตาร์ท
- **หลายภาษา**: หน้าการจัดการรองรับภาษาอังกฤษ จีนตัวย่อ และญี่ปุ่น
- **โหมดมืด**: หน้าจะปรับตามธีมของ LangBot โดยอัตโนมัติ

## คอมโพเนนต์

| คอมโพเนนต์ | ประเภท | คำอธิบาย |
|-----------|------|---------|
| `components/pages/manager/` | Page | อินเทอร์เฟซจัดการ FAQ (เพิ่ม แก้ไข ลบ ค้นหา) |
| `components/tools/search_faq.py` | Tool | ค้นหาคำสำคัญในรายการ FAQ เรียกใช้โดย LLM |
| `components/event_listener/default.py` | EventListener | ตัวรับฟังเหตุการณ์เริ่มต้น (ตัวยึดตำแหน่ง) |

## วิธีใช้

1. ติดตั้งปลั๊กอินใน LangBot
2. เปิดส่วน **หน้าปลั๊กอิน** ในแถบด้านข้างและเลือก **ตัวจัดการ FAQ**
3. เพิ่มคู่คำถาม-คำตอบผ่านหน้าเว็บ
4. เมื่อผู้ใช้ถามคำถามในการสนทนา LLM จะใช้เครื่องมือ `search_faq` เพื่อค้นหารายการ FAQ ที่ตรงกันและตอบตามนั้น
