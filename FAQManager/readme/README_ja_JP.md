# FAQ マネージャー

LangBot WebUI のビジュアルページで FAQ エントリを管理し、会話中に LLM が自動的に検索できるようにします。

## 機能

- **Page コンポーネント**：サイドバーの「プラグインページ」セクションから、質問と回答のペアを管理する CRUD インターフェースを提供。
- **Tool コンポーネント**：`search_faq` — LLM がキーワードで FAQ データベースを検索し、一致するエントリをユーザーに返します。
- **永続化ストレージ**：FAQ エントリはプラグインストレージに保存され、再起動後も保持されます。
- **多言語対応**：管理ページは英語、簡体字中国語、日本語に対応。
- **ダークモード**：ページは LangBot のテーマに自動的に適応します。

## コンポーネント

| コンポーネント | タイプ | 説明 |
|---|---|---|
| `components/pages/manager/` | Page | FAQ 管理 UI（追加、編集、削除、検索） |
| `components/tools/search_faq.py` | Tool | FAQ エントリのキーワード検索（LLM が呼び出し可能） |
| `components/event_listener/default.py` | EventListener | デフォルトイベントリスナー（プレースホルダー） |

## 使い方

1. LangBot にプラグインをインストールします。
2. サイドバーの **プラグインページ** セクションから **FAQ 管理** を選択します。
3. ページから質問と回答のペアを追加します。
4. ユーザーが会話で質問すると、LLM が `search_faq` ツールを使って一致する FAQ エントリを検索し、回答します。
