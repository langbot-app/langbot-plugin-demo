# 关键词监控

监控群聊消息中的关键词，命中时立即私聊通知管理员。

## 功能

- 🔔 实时监控群聊关键词
- 📋 支持多关键词配置（逗号分隔）
- 🎯 可指定监控群组或监控所有群
- 🤖 选择发送告警的机器人
- ⏱️ 冷却机制防止告警轰炸
- 🔤 可选大小写敏感匹配

## 使用方法

1. 配置需要监控的关键词（如 `bug,紧急,求助`）
2. 填写接收告警的管理员会话 ID
3. 当群消息命中关键词时，你会收到私聊告警通知

## 配置

| 选项 | 说明 | 默认值 |
|---|---|---|
| 关键词 | 逗号分隔的关键词列表 | （必填） |
| 群组 ID | 逗号分隔的群组 ID（留空 = 全部群） | 所有群 |
| 管理员会话 ID | 接收告警的用户 ID | （必填） |
| 告警机器人 | 发送告警的 bot | 第一个可用 |
| 大小写敏感 | 是否区分大小写 | 否 |
| 冷却时间 | 同群同关键词最短间隔秒数 | 60 |
