# 關鍵字提醒

監控群組聊天訊息中的特定關鍵字並接收即時私人訊息警報。

＃＃ 特徵

- 🔔 跨群聊即時關鍵字監控
- 📋可設定的關鍵字清單（以逗號分隔）
- 🎯 監控特定組或所有組
- 🤖 選擇哪個機器人發送警報
- ⏱️ 冷卻以防止警報垃圾郵件
- 🔤 可選區分大小寫的匹配

## 它是如何工作的

1. 設定要監控的關鍵字（例如`bug,urgent,help`）
2. 將您的使用者/會話 ID 設定為管理員以接收警報
3. 當有人在受監控的群組中傳送包含關鍵字的訊息時，您會收到包含完整上下文的私人訊息警報

＃＃ 配置

|選項 |說明 |預設 |
|---|---|---|
|關鍵字|要監控的以逗號分隔的關鍵字 | （必填）|
|組 ID |以逗號分隔的組 ID（空 = 全部）|所有團體|
|管理會話 ID |誰收到警報 | （必填）|
|警報機器人 |哪個機器人發送警報 |首次上市 |
|區分大小寫 |區分大小寫的匹配 |關閉 |
|冷卻時間|每組相同關鍵字警報之間的秒數 | 60|

## 警報格式

```
🔔 关键词告警
━━━━━━━━━━━━━━
关键词: urgent
群组: 123456789
发送者: 987654321
━━━━━━━━━━━━━━
Hey, this is urgent, the server is down!
```
