# การแจ้งเตือนคำหลัก

ตรวจสอบข้อความแชทกลุ่มสำหรับคำสำคัญเฉพาะ และรับการแจ้งเตือนข้อความส่วนตัวทันที

## คุณสมบัติ

- 🔔 การตรวจสอบคำหลักแบบเรียลไทม์ในการแชทกลุ่ม
- 📋 รายการคำหลักที่กำหนดค่าได้ (คั่นด้วยเครื่องหมายจุลภาค)
- 🎯 ติดตามเฉพาะกลุ่มหรือทุกกลุ่ม
- 🤖 เลือกบอทที่จะส่งการแจ้งเตือน
- ⏱️ คูลดาวน์เพื่อป้องกันสแปมการแจ้งเตือน
- 🔤 การจับคู่แบบคำนึงถึงตัวพิมพ์เล็กและใหญ่หรือไม่ก็ได้

## มันทำงานอย่างไร

1. กำหนดค่าคำหลักที่คุณต้องการตรวจสอบ (เช่น`bug,urgent,help`)
2. ตั้งค่า ID ผู้ใช้/เซสชันของคุณเป็นผู้ดูแลระบบเพื่อรับการแจ้งเตือน
3. เมื่อมีคนส่งข้อความที่มีคำสำคัญในกลุ่มที่ได้รับการตรวจสอบ คุณจะได้รับการแจ้งเตือนข้อความส่วนตัวพร้อมบริบททั้งหมด

## การกำหนดค่า

| ตัวเลือก | คำอธิบาย | ค่าเริ่มต้น |
|---|---|---|
| คำสำคัญ | คำหลักที่คั่นด้วยเครื่องหมายจุลภาคเพื่อติดตาม | (จำเป็น) |
| รหัสกลุ่ม | ID กลุ่มที่คั่นด้วยเครื่องหมายจุลภาค (ว่าง = ทั้งหมด) | ทุกกลุ่ม |
| รหัสเซสชันผู้ดูแลระบบ | ใครได้รับการแจ้งเตือน | (จำเป็น) |
| บอทแจ้งเตือน | บอทตัวไหนส่งการแจ้งเตือน | วางจำหน่ายครั้งแรก |
| คำนึงถึงขนาดตัวพิมพ์ | การจับคู่แบบคำนึงถึงตัวพิมพ์เล็กและตัวพิมพ์ใหญ่ | ปิด |
| คูลดาวน์ | วินาทีระหว่างการแจ้งเตือนคำหลักเดียวกันต่อกลุ่ม | 60 |
