# キーワードアラート

グループチャットのメッセージを監視し、キーワードが含まれる場合にプライベートメッセージで管理者に通知します。

## 機能

- 🔔 リアルタイムキーワード監視
- 📋 複数キーワード対応（カンマ区切り）
- 🎯 特定のグループまたは全グループを監視
- 🤖 アラート送信ボットを選択可能
- ⏱️ クールダウンでアラートスパムを防止
- 🔤 大文字小文字の区別オプション

## 設定

| オプション | 説明 | デフォルト |
|---|---|---|
| キーワード | カンマ区切りのキーワード | （必須） |
| グループID | カンマ区切り（空=全グループ） | 全グループ |
| 管理者ID | アラートを受信するユーザー | （必須） |
| ボット | アラートを送信するボット | 最初の利用可能なボット |
| 大文字小文字区別 | 大文字小文字を区別するか | オフ |
| クールダウン | 同じキーワードのアラート間隔（秒） | 60 |
