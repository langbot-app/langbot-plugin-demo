# GoogleSearch

Una herramienta para realizar búsquedas en Google SERP y extraer fragmentos y páginas web. La entrada debe ser una consulta de búsqueda.

## Uso

Instale el complemento, luego configure el `api_key` en la configuración del complemento.
Puede obtener el `api_key` de [SERP API](https://serpapi.com/dashboard).
