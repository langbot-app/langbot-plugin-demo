# TavilySearch Plugin

ปลั๊กอินสำหรับ [LangBot](https://github.com/langbot-app/langbot) ที่ให้ความสามารถในการค้นหาโดยใช้ Tavily API ซึ่งเป็นเครื่องมือค้นหาที่สร้างขึ้นสำหรับเอเจนซี่ AI (LLMs) โดยเฉพาะ

## คุณสมบัติ

- การค้นหาเว็บแบบเรียลไทม์ที่ขับเคลื่อนโดย Tavily
- รองรับความลึกในการค้นหาที่แตกต่างกัน (basic/advanced)
- การค้นหาเฉพาะหัวข้อ (general/news/finance)
- รวมคำตอบที่สร้างโดย AI
- รวมรูปภาพที่เกี่ยวข้อง
- รวมเนื้อหา HTML ดิบ
- กำหนดจำนวนผลลัพธ์ได้เอง

## การติดตั้ง

1. ติดตั้งปลั๊กอิน

2. กำหนดค่าคีย์ Tavily API ของคุณ:
   - รับคีย์ API ของคุณจาก [Tavily](https://tavily.com/)
   - เพิ่มคีย์ API ในการกำหนดค่าปลั๊กอินใน LangBot

## วิธีใช้งาน

ปลั๊กอินนี้จะเพิ่มเครื่องมือ `tavily_search` ที่ LLM สามารถใช้ในการสนทนาได้

### พารามิเตอร์

- **query** (จำเป็น): สตริงคำค้นหา
- **search_depth** (เลือกได้): "basic" (เริ่มต้น) หรือ "advanced"
- **topic** (เลือกได้): "general" (เริ่มต้น), "news" หรือ "finance"
- **max_results** (เลือกได้): จำนวนผลลัพธ์ (1-20, เริ่มต้น: 5)
- **include_answer** (เลือกได้): รวมคำตอบที่สร้างโดย AI (เริ่มต้น: false)
- **include_images** (เลือกได้): รวมรูปภาพที่เกี่ยวข้อง (เริ่มต้น: false)
- **include_raw_content** (เลือกได้): รวมเนื้อหา HTML ดิบ (เริ่มต้น: false)

### ตัวอย่าง

เมื่อแชทกับ LangBot ของคุณ LLM สามารถใช้เครื่องมือนี้ได้โดยอัตโนมัติ:

```
ผู้ใช้: ข่าวล่าสุดเกี่ยวกับปัญญาประดิษฐ์คืออะไร?

บอท: [ใช้เครื่องมือ tavily_search พร้อมกับ topic="news"]
```

## การพัฒนา

สำหรับการพัฒนาหรือแก้ไขปลั๊กอินนี้:

1. แก้ไขตรรกะของเครื่องมือใน `components/tools/tavily_search.py`
2. แก้ไขการกำหนดค่าใน `manifest.yaml`
3. อัปเดตพารามิเตอร์ของเครื่องมือใน `components/tools/tavily_search.yaml`

## การกำหนดค่า

ปลั๊กอินต้องการการกำหนดค่าดังต่อไปนี้:

- **tavily_api_key**: คีย์ Tavily API ของคุณ (จำเป็น)

## สัญญาอนุญาต

ปลั๊กอินนี้เป็นส่วนหนึ่งของระบบนิเวศปลั๊กอิน LangBot
