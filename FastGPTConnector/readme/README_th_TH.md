# FastGPTConnector

เรียกค้นความรู้จากฐานความรู้ FastGPT โดยใช้ FastGPT API

## เกี่ยวกับ FastGPT

FastGPT คือระบบถามตอบจากฐานความรู้แบบโอเพนซอร์สที่สร้างขึ้นจากโมเดล LLM ซึ่งให้ความสามารถในการประมวลผลข้อมูลและการเรียกใช้โมเดลที่พร้อมใช้งานทันทีสำหรับสถานการณ์การถามตอบที่ซับซ้อน

## คุณสมบัติ

- ค้นหาและเรียกคืนความรู้จากชุดข้อมูล/ฐานความรู้ของ FastGPT
- รองรับโหมดการค้นหาที่หลากหลาย (การค้นหาด้วยเวกเตอร์, การค้นหาแบบข้อความเต็ม, การค้นหาแบบผสม)
- กำหนดค่าเกณฑ์ความคล้ายคลึงและขีดจำกัดโทเค็นได้
- มีตัวเลือกการจัดลำดับใหม่ (re-ranking) เพื่อผลลัพธ์ที่ดีขึ้น
- เพิ่มประสิทธิภาพการสืบค้นด้วยโมเดลขยาย (extension models)

## การกำหนดค่า

ปลั๊กอินนี้ต้องการพารามิเตอร์การกำหนดค่าดังต่อไปนี้:

### พารามิเตอร์ที่จำเป็น

- **api_base_url**: Base URL สำหรับ FastGPT API
  - สำหรับการติดตั้งในเครื่อง: `http://localhost:3000` (ค่าเริ่มต้น)
  - สำหรับเซิร์ฟเวอร์ระยะไกล: URL เซิร์ฟเวอร์ของคุณ (เช่น `https://your-domain.com`)
- **api_key**: FastGPT API key ของคุณ
  - รูปแบบ: `fastgpt-xxxxx`
- **dataset_id**: ID ของฐานความรู้/ชุดข้อมูล FastGPT ของคุณ

### พารามิเตอร์เสริม

- **limit** (ค่าเริ่มต้น: 5000): จำนวนโทเค็นสูงสุดที่เรียกค้น
- **similarity** (ค่าเริ่มต้น: 0.0): คะแนนความคล้ายคลึงขั้นต่ำ (0-1)
- **search_mode** (ค่าเริ่มต้น: embedding): วิธีการค้นหาที่จะใช้
  - `embedding`: การค้นหาความหมายเชิงบริบทด้วยเวกเตอร์ (Semantic embedding search)
  - `fullTextRecall`: การค้นหาด้วยคำสำคัญแบบข้อความเต็ม (Full-text keyword search)
  - `mixedRecall`: การค้นหาแบบผสมที่รวมทั้งสองวิธีเข้าด้วยกัน
- **using_rerank** (ค่าเริ่มต้น: false): จะใช้การจัดลำดับใหม่หรือไม่
- **dataset_search_using_extension_query** (ค่าเริ่มต้น: false): จะใช้การเพิ่มประสิทธิภาพการสืบค้นหรือไม่
- **dataset_search_extension_model** (เลือกได้): โมเดลสำหรับการเพิ่มประสิทธิภาพการสืบค้น
- **dataset_search_extension_bg** (เลือกได้): คำอธิบายพื้นหลังสำหรับการเพิ่มประสิทธิภาพการสืบค้น

## วิธีรับค่าการกำหนดค่า

### การรับ FastGPT API Key ของคุณ

1. เข้าถึงอินสแตนซ์ FastGPT ของคุณ (เช่น `http://localhost:3000`)
2. ไปที่ส่วนการจัดการ API หรือการตั้งค่า
3. สร้างหรือคัดลอก API key ของคุณ (รูปแบบ: `fastgpt-xxxxx`)

### การรับ Dataset ID ของคุณ

1. ใน FastGPT ให้ไปที่รายการฐานความรู้ของคุณ
2. คลิกที่ฐานความรู้เพื่อดูรายละเอียด
3. สามารถพบ Dataset ID ได้ใน URL หรือหน้ารายละเอียดชุดข้อมูล

## การอ้างอิง API

ปลั๊กอินนี้ใช้ API ทดสอบการค้นหาชุดข้อมูลของ FastGPT:
- Endpoint: `POST /api/core/dataset/searchTest`
- เอกสารประกอบ: https://doc.fastgpt.io/docs/introduction/development/openapi/dataset

## วิธีการค้นหา

### การค้นหาด้วยเวกเตอร์ (Embedding Search)
ใช้ความคล้ายคลึงทางภาษาตามเวกเตอร์เวกเตอร์ เหมาะที่สุดสำหรับการทำความเข้าใจเจตนาในการสืบค้นและค้นหาเนื้อหาที่เกี่ยวข้องทางความหมาย

### การค้นหาแบบข้อความเต็ม (Full-Text Recall)
การค้นหาแบบข้อความเต็มตามคำสำคัญแบบดั้งเดิม เหมาะที่สุดสำหรับการค้นหาสิ่งที่ตรงกันทุกประการและคำศัพท์เฉพาะทาง

### การค้นหาแบบผสม (Mixed Recall)
รวมทั้งวิธีการค้นหาด้วยเวกเตอร์และแบบข้อความเต็มเข้าด้วยกัน ให้ผลลัพธ์ที่สมดุลทั้งการเข้าใจความหมายและการจับคู่คำสำคัญ
