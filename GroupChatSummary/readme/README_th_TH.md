#สรุปแชทกลุ่ม

สรุปข้อความแชทกลุ่มโดยใช้ LLM ไม่พลาดการสนทนาที่สำคัญอีกต่อไป

## คุณสมบัติ

-**การรวบรวมข้อความ**: บันทึกข้อความกลุ่มทั้งหมดโดยอัตโนมัติ
-**สรุปด้วยตนเอง**: ใช้คำสั่ง`!summary`เพื่อรับข้อมูลสรุปทันที
-**สรุปตามเวลา**: สรุปข้อความจาก N ชั่วโมงที่ผ่านมา
-**LLM Tool**: AI สามารถเรียกเครื่องมือสรุปได้เมื่อผู้ใช้ถามว่า "ฉันพลาดอะไรไป"
-**สรุปอัตโนมัติ**: เลือกเรียกข้อมูลสรุปหลังจากมีข้อความสะสม N รายการ
-**ที่เก็บข้อมูลถาวร**: ประวัติข้อความยังคงอยู่หลังจากรีสตาร์ทปลั๊กอิน

## คำสั่ง

| คำสั่ง | คำอธิบาย |
|---------|-------------|
|`!summary [นับ]`| สรุปข้อความ N ล่าสุด (ค่าเริ่มต้น: 100) |
|`!ชั่วโมงสรุป <N>`| สรุปข้อความจาก N ชั่วโมงที่ผ่านมา |
|`!สถานะสรุป`| แสดงสถานะบัฟเฟอร์ข้อความ |
|`!สรุปชัดเจน`| ล้างข้อความที่เก็บไว้ |

## การกำหนดค่า

| ตัวเลือก | ค่าเริ่มต้น | คำอธิบาย |
|--------|---------|-------------|
| ข้อความสูงสุด | 500 | ข้อความสูงสุดที่จัดเก็บต่อกลุ่ม |
| จำนวนสรุปเริ่มต้น | 100 | ข้อความที่จะสรุปตามค่าเริ่มต้น |
| สรุปอัตโนมัติ | ปิด | สรุปอัตโนมัติทุกข้อความ N |
| เกณฑ์สรุปอัตโนมัติ | 200 | ข้อความก่อนทริกเกอร์อัตโนมัติ |
| ภาษา | จีน | ภาษาเอาต์พุตสรุป |

## มันทำงานอย่างไร

1. ปลั๊กอินจะฟังข้อความกลุ่มทั้งหมดและจัดเก็บไว้ในหน่วยความจำ (ยังคงมีอยู่ในที่เก็บข้อมูล)
2. เมื่อทริกเกอร์ (คำสั่ง การเรียกใช้เครื่องมือ หรืออัตโนมัติ) จะจัดรูปแบบข้อความและส่งไปยัง LLM ที่กำหนดค่าของคุณ
3. LLM สร้างบทสรุปที่มีโครงสร้างพร้อมหัวข้อสำคัญ การตัดสินใจ และรายการดำเนินการ
