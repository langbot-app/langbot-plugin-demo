# GroupChatSummary

使用 LLM 总结群聊消息，再也不用担心错过重要讨论。

## 功能

- **消息收集**: 自动记录所有群聊消息
- **手动总结**: 使用 `!summary` 命令即时获取总结
- **按时间总结**: 总结最近 N 小时的消息
- **LLM 工具**: 当用户问"我错过了什么"时，AI 可以自动调用总结工具
- **自动总结**: 可选，每积累 N 条消息自动触发总结
- **持久存储**: 消息历史在插件重启后不会丢失

## 命令

| 命令 | 说明 |
|------|------|
| `!summary [数量]` | 总结最近 N 条消息（默认 100） |
| `!summary hours <N>` | 总结最近 N 小时的消息 |
| `!summary status` | 查看消息缓存状态 |
| `!summary clear` | 清空已存储的消息 |

## 配置

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 最大存储消息数 | 500 | 每个群最多存储的消息数 |
| 默认总结消息数 | 100 | 默认总结的消息数量 |
| 启用自动总结 | 关 | 每 N 条消息自动总结 |
| 自动总结阈值 | 200 | 触发自动总结的消息数 |
| 总结语言 | 简体中文 | 总结输出的语言 |
