# URL Summary

Automatically summarize web page content from URLs shared in chat.

When someone shares a link, the plugin fetches the page, extracts text content, and uses LLM to generate a concise summary.

## Features

- Auto-detects URLs in messages (both private and group chats)
- Extracts readable text from HTML pages
- Generates concise summaries using LLM
- Supports multiple languages (Chinese, English, Japanese)
- Configurable content length and model selection
- Handles up to 3 URLs per message
