# สรุป URL (URL Summary)

สรุปเนื้อหาหน้าเว็บโดยอัตโนมัติจาก URL ที่แชร์ในแชท

เมื่อมีคนแชร์ลิงก์ ปลั๊กอินจะดึงข้อมูลหน้าเว็บ แยกเนื้อหาข้อความ และใช้ LLM เพื่อสร้างสรุปที่รัดกุม

## คุณสมบัติ

- ตรวจจับ URL ในข้อความโดยอัตโนมัติ (ทั้งแชทส่วนตัวและแชทกลุ่ม)
- แยกข้อความที่อ่านได้จากหน้า HTML
- สร้างสรุปที่รัดกุมโดยใช้ LLM
- รองรับหลายภาษา (จีน, อังกฤษ, ญี่ปุ่น)
- กำหนดความยาวเนื้อหาและการเลือกโมเดลได้
- รองรับได้สูงสุด 3 URL ต่อข้อความ
