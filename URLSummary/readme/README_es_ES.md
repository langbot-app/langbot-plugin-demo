# Resumen de URL (URL Summary)

Resume automáticamente el contenido de las páginas web de las URL compartidas en el chat.

Cuando alguien comparte un enlace, el plugin obtiene la página, extrae el contenido de texto y utiliza LLM para generar un resumen conciso.

## Características

- Detecta automáticamente URLs en los mensajes (tanto en chats privados como grupales)
- Extrae texto legible de páginas HTML
- Genera resúmenes concisos utilizando LLM
- Soporta múltiples idiomas (Chino, Inglés, Japonés)
- Configurable en longitud de contenido y selección de modelo
- Maneja hasta 3 URLs por mensaje
