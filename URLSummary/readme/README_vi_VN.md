# Tóm tắt URL (URL Summary)

Tự động tóm tắt nội dung trang web từ các URL được chia sẻ trong cuộc trò chuyện.

Khi ai đó chia sẻ một liên kết, plugin sẽ tìm nạp trang, trích xuất nội dung văn bản và sử dụng LLM để tạo một bản tóm tắt ngắn gọn.

## Tính năng

- Tự động phát hiện URL trong tin nhắn (cả trò chuyện riêng tư và nhóm)
- Trích xuất văn bản có thể đọc được từ các trang HTML
- Tạo bản tóm tắt ngắn gọn bằng LLM
- Hỗ trợ nhiều ngôn ngữ (Trung Quốc, Anh, Nhật Bản)
- Có thể định cấu hình độ dài nội dung và lựa chọn mô hình
- Xử lý tối đa 3 URL cho mỗi tin nhắn
