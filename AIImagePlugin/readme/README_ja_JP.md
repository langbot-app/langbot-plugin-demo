# AI画像生成プラグイン

## 概要

OpenAI形式の画像生成APIに対応したAIイラストプラグインです。OpenAI画像生成API互換のサービスならどれでも利用できます。

## 主な機能

- ✅ OpenAI画像生成API形式に完全対応
- 🎨 APIエンドポイントやモデル名をカスタマイズ可能
- 📐 様々な画像サイズ・アスペクト比に対応
- 🔧 柔軟な設定オプション

## 設定方法

### API設定

- **APIエンドポイント**：デフォルトは `https://api.qhaigc.net` ですが、OpenAI形式互換の任意のAPIアドレスを利用可能です
- **APIキー**：[https://api.qhaigc.net/console/token](https://api.qhaigc.net/console/token) から取得してください
- **モデル名**：任意のモデル名を指定可能。デフォルトは `qh-draw-x1-pro`

### 画像サイズオプション

下記の画像サイズに対応しています：

- 正方形 1:1 (1024x1024)
- 正方形 1:1 (1280x1280)
- 縦型 3:5 (768x1280)
- 横型 5:3 (1280x768)
- 縦型 9:16 (720x1280)
- 横型 16:9 (1280x720)
- 横型 4:3 (1024x768)
- 縦型 3:4 (768x1024)

## 使い方

`!draw` コマンドで画像を生成：

```bash
# 画像を生成
!draw 美しい夕日の風景
!draw 虹の上に座る猫
```

## インストール手順

1. LangBotのプラグイン管理ページから本プラグインをインストール
2. APIキーを取得: [https://api.qhaigc.net/console/token](https://api.qhaigc.net/console/token)
3. プラグインの設定画面でAPIキーを入力
4. APIアドレス・モデル名・デフォルト画像サイズを任意で設定（オプション）

## 対応モデル

### Nano Bananaシリーズ

- **Nano Banana 1**（2025年8月リリース）：Google DeepMindによる画像生成モデル。Gemini 2.5 Flashアーキテクチャをベースにし、パラメータ数は4.5億〜80億。キャラクターの一貫性、複数画像の融合、部分編集が得意。LMArena画像編集リーダーボードで1362点を記録し、ECやデザイン、教育等で幅広く活用されています。

- **Nano Banana 2**（2025年11月リリース）：初代モデルの全面的アップグレード。ネイティブ2K解像度や4K超解像度も選択可能。生成速度は初代比で300%向上し、複雑なシーンも10秒で生成。中国語テキスト描画や数式変形も大幅強化。物理的・知識理解力も強化し、「認知＋生成」のハイブリッド構成でクリエイティブ業界の生産性を革新。

### 启航AI绘图（Qihang AI Drawing）シリーズ

- **qh-draw-3d**：流行の3Dスタイル画像生成に特化。精巧な3Dモデル・明瞭な3D表現が特徴です。
- **qh-draw-4d**：流行の4Dスタイル画像生成に特化。精巧な4Dモデルと、リアルに近いビジュアル（実写は生成しません）。
- **qh-draw-x1-pro**：启航AI绘图 x1-proモデル。SDオープンソースモデルをベースとし、自然言語理解機能も備えます。
- **qh-draw-x2-preview**：自社開発のプロフェッショナル描画モデルV2.0。x1-proの言語理解・総合描画能力を大幅に強化。
- **qh-draw:韓国漫画風**：クラシックな韓国コミック風画像生成に特化。鮮やかなカラーと滑らかな線、場面・雰囲気も忠実に再現。

### 利用可能モデル一覧

`nano-banana-1`, `nano-banana-2`, `qh-draw-3d`, `qh-draw-4d`, `qh-draw-x1-pro`, `qh-draw-x2-preview`, `qh-draw:韓国漫画風`

## 互換性

本プラグインはOpenAI画像生成API仕様に準拠する全てのサービスで利用可能です（例: OpenAI DALL-E 3など）。

- OpenAI DALL-E 3
- OpenAI互換のその他画像生成サービス
