# AI画像生成プラグイン

## 概要

OpenAI形式と互換性のある画像生成プラグインで、OpenAI画像生成APIに準拠する任意のサービスをサポートしています。

## 機能

- OpenAI画像生成API形式と完全互換
- カスタムAPIエンドポイントとモデル名のサポート
- 複数の画像アスペクト比オプション
- 柔軟な設定オプション

## 設定

### API設定

- **APIエンドポイント**：デフォルトは `https://api.qhaigc.net`、他のOpenAI互換APIエンドポイントにカスタマイズ可能
- **APIキー**：[https://api.qhaigc.net/console/token](https://api.qhaigc.net/console/token) から取得
- **モデル名**：カスタムモデル名を入力可能、デフォルトは `qh-draw-x1-pro`

### 画像サイズオプション

以下の画像サイズをサポートしています：

- 正方形 1:1 (1024x1024)
- 正方形 1:1 (1280x1280)
- 縦長 3:5 (768x1280)
- 横長 5:3 (1280x768)
- 縦長 9:16 (720x1280)
- 横長 16:9 (1280x720)
- 横長 4:3 (1024x768)
- 縦長 3:4 (768x1024)

## 使用方法

`!draw` コマンドを使用して画像を生成します：

```bash
# 画像を生成
!draw 美しい夕日の風景
!draw 虹の上に座っている猫
```

## インストール手順

1. LangBotプラグイン管理ページから本プラグインをインストール
2. APIキーを取得：[https://api.qhaigc.net/console/token](https://api.qhaigc.net/console/token)
3. プラグイン設定にAPIキーを入力
4. APIエンドポイント、モデル名、デフォルト画像サイズを設定（オプション）

## 互換性

このプラグインは、OpenAI画像生成API仕様に準拠する任意のサービスと互換性があります。以下を含みますがこれらに限定されません：

- OpenAI DALL-E 3
- その他のOpenAI互換画像生成サービス
