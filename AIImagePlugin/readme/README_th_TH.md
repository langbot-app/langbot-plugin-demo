# ปลั๊กอินการสร้างภาพ AI

## การแนะนำ

ปลั๊กอินการวาดภาพที่เข้ากันได้กับรูปแบบ API การสร้างภาพ OpenAI รองรับบริการใดๆ ที่เข้ากันได้กับ API การสร้างอิมเมจ OpenAI

## คุณสมบัติ

- ✅เข้ากันได้กับรูปแบบ API การสร้างภาพ OpenAI อย่างสมบูรณ์
- 🎨 รองรับจุดสิ้นสุด API ที่กำหนดเองและชื่อรุ่น
- 📐 รองรับอัตราส่วนภาพหลายขนาด
- Optim ตัวเลือกการกำหนดค่าที่ยืดหยุ่น

## คู่มือการกำหนดค่า

### การกำหนดค่า API

-**จุดสิ้นสุด API**: มีค่าเริ่มต้นเป็น`https://api.qhaigc.net`ซึ่งสามารถปรับแต่งให้เป็นจุดสิ้นสุด API สไตล์ OpenAI ที่เข้ากันได้
-**คีย์ API**: รับคีย์ API ของคุณจาก [https://api.qhaigc.net/console/token](https://api.qhaigc.net/console/token)
-**ชื่อรุ่น**: คุณสามารถตั้งชื่อรุ่นแบบกำหนดเองได้ โดยค่าเริ่มต้นคือ`qh-draw-x1-pro`

### ตัวเลือกขนาดภาพ

รองรับขนาดรูปภาพต่อไปนี้:

- สี่เหลี่ยมจัตุรัส 1:1 (1024x1024)
- สี่เหลี่ยมจัตุรัส 1:1 (1280x1280)
- แนวตั้ง 3:5 (768x1280)
- แนวนอน 5:3 (1280x768)
- แนวตั้ง 9:16 (720x1280)
- แนวนอน 16:9 (1280x720)
- แนวนอน 4:3 (1024x768)
- แนวตั้ง 3:4 (768x1024)

## การใช้งาน

ใช้คำสั่ง`!draw`เพื่อสร้างภาพ:

```bash
# Generate an image
!draw a beautiful sunset landscape
!draw a cat sitting on a rainbow
```

## ขั้นตอนการติดตั้ง

1. ติดตั้งปลั๊กอินนี้จากหน้าการจัดการปลั๊กอิน LangBot
2. รับคีย์ API ของคุณ: [https://api.qhaigc.net/console/token](https://api.qhaigc.net/console/token)
3. กรอกคีย์ API ในการกำหนดค่าปลั๊กอิน
4. (ไม่บังคับ) กำหนดค่าตำแหน่งข้อมูล API ชื่อรุ่น และขนาดรูปภาพเริ่มต้น

## รุ่นที่รองรับ

### ซีรี่ย์กล้วยนาโน

-**Nano Banana 1**(เปิดตัวเดือนสิงหาคม 2025): โมเดลการสร้างรูปภาพโดย Google DeepMind อิงตามสถาปัตยกรรม Gemini 2.5 Flash พร้อมพารามิเตอร์ 450M ถึง 8B จุดแข็งหลักคือความสม่ำเสมอของบทบาท การผสมผสานหลายภาพ และการแก้ไขเฉพาะที่ เป็นผู้นำในกระดานผู้นำการแก้ไขรูปภาพ LMArena ด้วยคะแนน 1,362 และมีการใช้กันอย่างแพร่หลายในอีคอมเมิร์ซ การออกแบบ การศึกษา และอื่นๆ

-**Nano Banana 2**(เปิดตัวเดือนพฤศจิกายน 2568): การอัปเกรดที่ครอบคลุมของรุ่นแรก รองรับความละเอียดเนทิฟ 2K พร้อมตัวเลือกความละเอียดสูงพิเศษ 4K ความเร็วในการสร้างได้รับการปรับปรุงขึ้น 300% ทำให้สามารถใช้งานฉากที่ซับซ้อนได้ในเวลาเพียง 10 วินาที ความก้าวหน้าครั้งสำคัญในการแสดงข้อความภาษาจีนและการสืบค้นสูตรทางคณิตศาสตร์ เข้าใจตรรกะทางกายภาพและความรู้ของโลก โดยใช้สถาปัตยกรรมไฮบริด "ความรู้ความเข้าใจ + การสร้าง" ที่ปฏิวัติประสิทธิภาพการผลิตสำหรับอุตสาหกรรมสร้างสรรค์

### ซีรีส์การวาด AI ของ Qihang

-**qh-draw-3d**: มุ่งเน้นไปที่การสร้างภาพสไตล์ 3 มิติยอดนิยม โดดเด่นด้วยโมเดล 3 มิติที่ละเอียดอ่อนและเอฟเฟกต์ภาพ 3 มิติที่แข็งแกร่ง
-**qh-draw-4d**: มุ่งเน้นไปที่การสร้างภาพสไตล์ 4D ยอดนิยม ด้วยโมเดล 4D ที่ซับซ้อนและภาพที่ใกล้เคียงกับความเป็นจริง แต่ไม่ใช่ภาพถ่ายจริง
-**qh-draw-x1-pro**: โมเดล Qihang AI Drawing x1-pro อิงตามโมเดล SD แบบโอเพ่นซอร์สพร้อมความเข้าใจภาษาที่เป็นธรรมชาติ
-**qh-draw-x2-preview**: โมเดลการวาดภาพมืออาชีพที่พัฒนาตนเอง V2.0 บนพื้นฐานของ x1-pro ช่วยเพิ่มความเข้าใจภาษาและความสามารถในการวาดภาพที่ครอบคลุม ทำให้เหมาะสำหรับงานต่างๆ มากขึ้น
-**qh-draw:สไตล์การ์ตูนเกาหลี**: เชี่ยวชาญในการสร้างภาพการ์ตูนเกาหลี 2 มิติคลาสสิก โดยมีสีสันสดใส เส้นเรียบ และอารมณ์ฉากที่แม่นยำสำหรับการ์ตูนเกาหลี
