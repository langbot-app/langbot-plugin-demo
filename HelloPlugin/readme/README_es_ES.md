# HolaPlugin

Este complemento es un complemento experimental que proporciona demostraciones sobre los componentes`EventListener`,`Tool`y`Command`.

-`components/event_listener/default.py`: simplemente responde un mensaje cuando se recibe un mensaje personal.
-`components/tools/get_weather_alerts.py`: proporciona una herramienta que se puede utilizar para obtener alertas meteorológicas de los estados de EE. UU.
-`components/commands/info.py`: responde al comando`info`con el contexto de ejecución del comando.
