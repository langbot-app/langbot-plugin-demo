# HelloPlugin

このプラグインは、`EventListener`、`Tool`、`Command` コンポーネントのデモを提供する実験的なプラグインです。

- `components/event_listener/default.py`：個人メッセージを受信したときに、シンプルにメッセージを返信します。
- `components/tools/get_weather_alerts.py`：米国の州の気象警報を取得するために使用できるツールを提供します。
- `components/commands/info.py`：`info` コマンドに応答し、コマンド実行のコンテキストを返します。
