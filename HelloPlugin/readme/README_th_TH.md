# สวัสดีปลั๊กอิน

ปลั๊กอินนี้เป็นปลั๊กอินทดลองที่ให้การสาธิตเกี่ยวกับส่วนประกอบ`EventListener`,`Tool`และ`Command`

-`components/event_listener/default.py`: เพียงตอบกลับข้อความเมื่อได้รับข้อความถึงบุคคล
-`components/tools/get_weather_alerts.py`: จัดเตรียมเครื่องมือที่สามารถใช้เพื่อรับการแจ้งเตือนสภาพอากาศของรัฐในสหรัฐฯ
-`components/commands/info.py`: ตอบสนองต่อคำสั่ง`info`ด้วยบริบทของการดำเนินการคำสั่ง
