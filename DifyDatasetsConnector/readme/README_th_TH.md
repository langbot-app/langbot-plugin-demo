# DifyDatasetsConnector

ดึงความรู้จากหรือจัดเก็บไฟล์ลงในฐานความรู้ของ Dify โดยใช้ Dify API

## การกำหนดค่า

โปรดเพิ่มฐานความรู้ภายนอกใน LangBot และเลือก "DifyDatasetsConnector" เป็นประเภทการดึงความรู้

### การตั้งค่าการสร้าง (ตั้งค่าเมื่อสร้างฐานความรู้)

-**api_base_url**: URL ฐานสำหรับ Dify API
  - สำหรับ Dify Cloud:`https://api.dify.ai/v1`(ค่าเริ่มต้น)
  - สำหรับอินสแตนซ์ที่โฮสต์เอง: URL เซิร์ฟเวอร์ของคุณ (เช่น`http://localhost/api`หรือ`https://your-domain.com/api`)
-**dify_apikey**: คีย์ Dify API ของคุณจากอินสแตนซ์ Dify ของคุณ
-**dataset_id**: ID ของฐานความรู้/ชุดข้อมูล Dify ของคุณ

### การตั้งค่าการดึงข้อมูล (ปรับได้ตามแบบสอบถาม)

-**search_method**(ค่าเริ่มต้น: semantic_search): วิธีค้นหาที่จะใช้
  -`keyword_search`: การค้นหาตามคำหลัก
  -`semantic_search`: การค้นหาความคล้ายคลึงทางความหมาย (ค่าเริ่มต้น)
  -`full_text_search`: ค้นหาข้อความแบบเต็ม
  -`hybrid_search`: การค้นหาแบบไฮบริดที่รวมความหมายและข้อความแบบเต็ม
-**top_k**(ค่าเริ่มต้น: 5): จำนวนผลลัพธ์ที่ดึงได้สูงสุด
-**score_threshold_enabled**(ค่าเริ่มต้น: false): ว่าจะเปิดใช้งานการกรองเกณฑ์คะแนนหรือไม่
-**score_threshold**(ค่าเริ่มต้น: 0.5): คะแนนความเกี่ยวข้องขั้นต่ำ (0-1) จะแสดงเฉพาะเมื่อมีการเปิดใช้งานเกณฑ์คะแนน
-**reranking_enable**(ค่าเริ่มต้น: false): เปิดใช้งานการจัดอันดับใหม่เพื่อปรับปรุงคุณภาพผลลัพธ์ โมเดลการจัดอันดับใหม่จะถูกดึงมาจากการตั้งค่าชุดข้อมูล Dify ของคุณโดยอัตโนมัติ โปรดกำหนดค่าโมเดลการจัดอันดับใหม่ในคอนโซล Dify ก่อน

## วิธีรับค่าการกำหนดค่า

### รับคีย์ Dify API ของคุณ

1. ไปที่ https://cloud.dify.ai/
2. ไปที่หน้าฐานความรู้ของคุณ
3. คลิกที่ "API ACCESS" ในแถบด้านข้างซ้าย
4. สร้างหรือคัดลอกคีย์ API ของคุณจากส่วน "คีย์ API"

### รับรหัสชุดข้อมูลของคุณ

1. ในรายการฐานความรู้ Dify ให้คลิกที่ฐานความรู้ของคุณ
2. รหัสชุดข้อมูลอยู่ใน URL:`https://cloud.dify.ai/datasets/{dataset_id}`
3. หรือคุณสามารถค้นหาได้ในหน้าเอกสารประกอบ API ของฐานความรู้ของคุณ

### การกำหนดค่าการจัดอันดับใหม่

1. ในคอนโซล Dify ให้ไปที่การตั้งค่าชุดข้อมูลของคุณ
2. เปิดใช้งานการจัดอันดับใหม่และเลือกโมเดลการจัดอันดับใหม่ (เช่น`cohere/rerank-v3.5`)
3. บันทึกการตั้งค่า
4. ใน LangBot เปิดใช้งานการสลับ "เปิดใช้งานการจัดอันดับใหม่" — ปลั๊กอินจะใช้โมเดลที่กำหนดค่าใน Dify โดยอัตโนมัติ

## การอ้างอิง API

ปลั๊กอินนี้ใช้ Dify Dataset API:
- การดึงข้อมูล:`POST /v1/datasets/{dataset_id}/retrieve`
- ข้อมูลชุดข้อมูล:`GET /v1/datasets/{dataset_id}`
- การอัปโหลดเอกสาร:`POST /v1/datasets/{dataset_id}/document/create-by-file`
- ลบเอกสาร:`DELETE /v1/datasets/{dataset_id}/documents/{document_id}`
- เอกสารประกอบ: https://docs.dify.ai/
