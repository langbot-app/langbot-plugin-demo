# 自動翻譯

自動偵測群組聊天中的外語訊息並使用 LLM 進行翻譯。

＃＃ 特徵

- 🌐 自動偵測訊息語言 — 無需手動指令
- 🤖 使用法學碩士進行自然、高品質的翻譯
- ⚙️可設定的目標語言（中文、英文、日文、韓文、法文、西班牙文）
- 📏 跳過簡訊、表情符號和 URL
- 🔇 僅在需要時進行翻譯 — 相同語言的訊息將被忽略
- 👥 預設僅限群組，可選私人聊天支持

## 它是如何工作的

1. 當在群組聊天中收到訊息時，外掛程式會將文字傳送給 LLM
2. LLM 偵測訊息的語言是否與配置的目標語言不同
3. 如果需要翻譯，插件會回覆帶有 🌐 前綴的翻譯文本
4. 已經使用目標語言的訊息將被默默忽略

＃＃ 配置

|選項 |說明 |預設 |
|---|---|---|
|目標語言 |翻譯成的語言 |中文（簡體）|
|法學碩士模式|用於翻譯的模型 |首次上市 |
|最小文字長度 |跳過比這短的訊息 | 4 個字元 |
|在私人聊天中啟用 |也翻譯私訊 |關閉 |
