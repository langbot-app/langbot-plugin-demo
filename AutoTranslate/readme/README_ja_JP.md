# 自動翻訳

グループチャットの外国語メッセージを自動検出し、LLMで高品質な翻訳を行います。

## 機能

- 🌐 メッセージの言語を自動検出、コマンド不要
- 🤖 LLMによる自然で高品質な翻訳
- ⚙️ ターゲット言語を設定可能（中国語、英語、日本語、韓国語、フランス語、スペイン語）
- 📏 短いメッセージ、絵文字、URLは自動スキップ
- 🔇 同じ言語のメッセージは無視
- 👥 デフォルトはグループチャットのみ、プライベートチャットも有効化可能

## 設定

| オプション | 説明 | デフォルト |
|---|---|---|
| ターゲット言語 | 翻訳先の言語 | 中国語（簡体字） |
| LLMモデル | 翻訳に使用するモデル | 最初の利用可能なモデル |
| 最小テキスト長 | これより短いメッセージをスキップ | 4文字 |
| プライベートチャットで有効化 | DMでも翻訳を行う | オフ |
