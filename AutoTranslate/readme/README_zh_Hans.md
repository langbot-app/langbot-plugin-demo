# 自动翻译

自动检测群聊中的外语消息，并使用 LLM 进行高质量翻译。

## 功能特点

- 🌐 自动检测消息语言，无需手动命令
- 🤖 使用 LLM 翻译，质量远超传统翻译 API
- ⚙️ 可配置目标语言（中文、英文、日文、韩文、法文、西班牙文）
- 📏 自动跳过短消息、表情、URL
- 🔇 同语言消息静默忽略，不打扰
- 👥 默认仅群聊启用，可选开启私聊翻译

## 工作原理

1. 群聊收到消息时，插件将文本发送给 LLM 进行语言检测
2. 如果消息不是目标语言，LLM 会进行翻译
3. 翻译结果以 🌐 前缀回复
4. 已经是目标语言的消息会被静默跳过

## 配置项

| 选项 | 说明 | 默认值 |
|---|---|---|
| 目标语言 | 翻译成的语言 | 简体中文 |
| LLM 模型 | 用于翻译的模型 | 第一个可用模型 |
| 最短文本长度 | 跳过短于此长度的消息 | 4 字符 |
| 私聊中启用 | 是否在私聊中也翻译 | 关闭 |
