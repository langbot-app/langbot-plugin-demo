# แปลอัตโนมัติ

ตรวจจับข้อความภาษาต่างประเทศในการแชทกลุ่มโดยอัตโนมัติและแปลโดยใช้ LLM

## คุณสมบัติ

- 🌐 ตรวจจับภาษาของข้อความอัตโนมัติ — ไม่ต้องใช้คำสั่งด้วยตนเอง
- 🤖 ใช้ LLM เพื่อการแปลที่เป็นธรรมชาติและมีคุณภาพสูง
- ⚙️ ภาษาเป้าหมายที่กำหนดค่าได้ (จีน อังกฤษ ญี่ปุ่น เกาหลี ฝรั่งเศส สเปน)
- 📏 ข้ามข้อความสั้น อีโมติคอน และ URL
- 🔇 แปลเมื่อจำเป็นเท่านั้น — ข้อความภาษาเดียวกันจะถูกละเว้น
- 👥 โดยค่าเริ่มต้น เฉพาะกลุ่มเท่านั้น รองรับการแชทส่วนตัวเพิ่มเติม

## มันทำงานอย่างไร

1. เมื่อได้รับข้อความในการแชทเป็นกลุ่ม ปลั๊กอินจะส่งข้อความไปยัง LLM
2. LLM ตรวจพบว่าข้อความเป็นภาษาอื่นนอกเหนือจากเป้าหมายที่กำหนดค่าไว้หรือไม่
3. หากจำเป็นต้องแปล ปลั๊กอินจะตอบกลับด้วยข้อความที่แปลนำหน้าด้วย 🌐
4. ข้อความในภาษาเป้าหมายอยู่แล้วจะถูกละเว้นอย่างเงียบๆ

## การกำหนดค่า

| ตัวเลือก | คำอธิบาย | ค่าเริ่มต้น |
|---|---|---|
| ภาษาเป้าหมาย | ภาษาที่จะแปลเป็น | จีน (ตัวย่อ) |
| รุ่น LLM | รุ่นที่จะใช้ในการแปล | วางจำหน่ายครั้งแรก |
| ความยาวข้อความขั้นต่ำ | ข้ามข้อความที่สั้นกว่านี้ | 4 ตัวอักษร |
| เปิดใช้งานในการแชทส่วนตัว | แปลข้อความส่วนตัวด้วย | ปิด |
